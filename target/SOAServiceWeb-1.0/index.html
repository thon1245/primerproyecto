Servicios SOA

Cuenta con los siguientes metodos:

Metodo Registrar

POST: http://localhost:8080/SOAServiceWeb/api/auth/register

Content-Type: application/json

{"username":"juan","password":"1234","role":"admin"}

Metodo Autenticar

POST: http://localhost:8080/SOAServiceWeb/api/auth/login

Content-Type: application/json

{"username":"juan","password":"1234","role":"admin"}

Metodo Saludo:

GET: http://localhost:8080/SOAServiceWeb/api/greetings/hello

Header PARAM:

user: juan
